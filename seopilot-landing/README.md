# SEOPilot Landing Page

Marketing landing page for SEOPilot — AI-Powered SEO for Shopify.

## Tech Stack

- Pure HTML + Tailwind CSS (CDN)
- No build step required
- Responsive design (mobile + desktop)

## Sections

1. **Hero** — Product tagline + CTA + interactive dashboard mockup
2. **Features** — 4 feature highlight cards
3. **Social Proof** — Key metrics (6 rules, <2s AI, 1-click fix)
4. **How It Works** — 3-step flow
5. **Pricing** — Free / Pro ($19) / Business ($39)
6. **CTA** — Final install call-to-action
7. **Footer** — Links + branding

## Run Locally

```bash
python3 -m http.server 3000
# Open http://localhost:3000
```
