# Agent-Hub Test Projects

Projects developed by AI agent teams using [Agent-Hub](https://github.com/huayang/agents), a multi-agent collaboration system.

Each project was built by a team of 3 agents (Product Manager + Developer + QA Engineer) working autonomously with zero manual intervention.
