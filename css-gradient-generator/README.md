# CSS Gradient Generator

**Live Demo**: https://css-gradient-generator-ruddy.vercel.app

**Deploy Status**: Deployed to Vercel

## Features

### Core Builder (M1)
- Real-time gradient preview (300x200+ px)
- Color stop management (add/remove/reorder with color picker + position slider)
- Linear and radial gradient type switching
- Angle control (0-360° slider) for linear, position selector (9 options) for radial
- Live CSS code output in copy-paste format
- One-click copy with "Copied!" visual feedback

### Enhanced Features (M2)
- **Preset Gallery**: 12 beautiful presets (Sunset, Ocean, Forest, Fire, Aurora, etc.) with thumbnails
- **Fullscreen Preview**: Full-window gradient preview with Esc to exit
- **Undo/Redo**: Full state history with Ctrl+Z / Ctrl+Shift+Z keyboard shortcuts
- **Direction Wheel**: Visual circular angle picker with drag interaction, synced with slider
- **Responsive Design**: Desktop dual-column / tablet+mobile single-column
- **Dark Theme**: WCAG AA compliant dark UI

## Tech Stack
- HTML5 + CSS3 + Vanilla JavaScript (ES6 Modules)
- No frameworks, no dependencies
- Static deployment on Vercel
