# Typing Speed Test - PRD

## 项目概述

一个纯前端的打字速度测试 Web 应用，用户可以通过打字练习来测量自己的打字速度（WPM）和准确率。

- **技术栈**: HTML + CSS + JavaScript（无框架依赖）
- **部署**: Vercel 静态部署
- **在线演示**: https://typing-test-drab.vercel.app
- **代码目录**: `projects/typing-test/`

## Milestone 列表

### Milestone 1: 基础打字测试 [已完成 ✅]

**目标**: 实现完整可用的基础打字速度测试功能。

**功能需求**:
1. **文本展示**: 显示一段随机英文文本供用户打字，文本库至少包含 10 段不同文本
2. **实时逐字符高亮**:
   - 已正确输入的字符: 绿色
   - 输入错误的字符: 红色背景
   - 当前待输入字符: 下划线标记
   - 未输入的字符: 默认灰色
3. **WPM 实时计算**: 每秒更新一次，WPM = (正确字符数 / 5) / 经过分钟数
4. **准确率显示**: 准确率 = 正确字符数 / 总输入字符数 * 100%
5. **60 秒倒计时**: 首次按键时开始计时，倒计时归零后自动结束测试
6. **结果展示**: 测试结束后显示最终成绩面板（WPM、准确率、总字符数）
7. **重新开始**: 提供重新开始按钮，点击后换新文本、重置所有状态
8. **输入方式**: 用户在页面聚焦状态下直接打字，无需点击输入框（隐藏 input 实现）

**UI/UX 要求**:
- 响应式设计，桌面和移动端均可正常使用
- 简洁现代的界面风格，深色/浅色主题均可
- 打字区域居中，字体大小适中（建议 1.2rem+），行间距舒适
- 统计数据（WPM、准确率、倒计时）始终可见

**验收标准**:
- [ ] 页面加载后显示随机文本，光标在第一个字符
- [ ] 按下第一个键后倒计时开始
- [ ] 每个字符输入后立即显示正确/错误高亮
- [ ] WPM 和准确率实时更新
- [ ] 60 秒结束后显示成绩面板，输入被禁用
- [ ] 点击重新开始后完全重置
- [ ] 移动端可正常使用

### Milestone 2: 增强体验 [已完成 ✅]

**目标**: 在基础功能上增加个性化和数据分析功能。

**功能需求**:
1. **多种时间选项**: 15s / 30s / 60s / 120s 可选
2. **历史成绩记录**: 使用 localStorage 存储，显示最近 10 次成绩（包含 WPM、准确率、日期）
3. **实时 WPM 折线图**: 每秒记录一个 WPM 数据点，测试结束后展示折线图
4. **错误字符统计**: 记录每个字母的错误次数，测试结束后展示"最易错字母 Top 5"
5. **多种文本类型**: 提供切换选项 - 常用英文 / 编程代码 / 名言警句
6. **打字音效**: 按键音效开关，提供至少 2 种音效可选

**验收标准**:
- [ ] 可切换不同时间长度，切换后自动重置
- [ ] 历史记录在 localStorage 中正确存储和读取
- [ ] 折线图准确反映每秒 WPM 变化
- [ ] 错误统计正确显示
- [ ] 不同文本类型内容有明显区别
- [ ] 音效可正常播放和静音
