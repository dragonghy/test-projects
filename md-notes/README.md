# Markdown Notes - 产品需求文档 (PRD)

**在线演示**: https://md-notes-iota.vercel.app

**部署状态**: ✅ 已部署到 Vercel

## 产品目标

为喜欢用 Markdown 写笔记的开发者和知识工作者提供一个轻量、快速、无需后端的笔记应用。用户打开网页即可开始写作，所有数据保存在浏览器本地。

## 项目信息

- **项目类型**：纯前端 Web 应用（HTML + CSS + JS，无框架）
- **部署方式**：Vercel 静态部署
- **代码目录**：`projects/md-notes/`
- **文件结构**：`index.html`, `style.css`, `app.js`

## 目标用户

喜欢用 Markdown 写笔记的开发者和知识工作者。

## 核心体验

打开应用后看到三栏布局：左侧笔记列表、中间 Markdown 编辑器、右侧实时预览。用户可以创建多个笔记，在编辑器中写 Markdown 内容，右侧立即渲染为 HTML。所有笔记保存在 localStorage。

## Milestone 1: 基础编辑功能

**状态**: 已完成 (2026-03-16 验收通过)

### 功能清单

#### F1.1 三栏布局
- 左侧：笔记列表侧边栏（约 250px 宽）
- 中间：Markdown 编辑器（等宽字体 textarea）
- 右侧：实时预览面板（渲染后的 HTML）
- 三栏自适应填满视口高度和宽度

#### F1.2 笔记 CRUD
- **创建**：点击侧边栏顶部的"新建笔记"按钮，创建新笔记并自动选中
- **重命名**：双击笔记标题可编辑名称
- **删除**：每个笔记项有删除按钮，点击后弹出确认对话框（confirm）再删除
- **选中**：点击笔记项切换到该笔记，编辑器和预览同步更新

#### F1.3 笔记列表排序
- 按最后修改时间降序排列（最近修改的在最上面）
- 每次编辑内容或重命名后自动更新排序

#### F1.4 Markdown 实时渲染
- 编辑器输入时，右侧预览面板实时更新
- 支持的 Markdown 语法：标题（h1-h6）、粗体、斜体、链接、图片、代码块（含语法高亮）、行内代码、有序列表、无序列表、引用、分割线、表格
- 可以使用 CDN 引入的 Markdown 解析库（如 marked.js）

#### F1.5 编辑器行号显示
- 编辑器左侧显示行号
- 行号随内容滚动同步

#### F1.6 自动保存
- 用户停止输入 500ms 后自动保存到 localStorage
- 保存时显示简短的保存状态提示（如"已保存"）

#### F1.7 默认欢迎笔记
- 首次打开应用（localStorage 无数据）时，自动创建一个"欢迎"笔记
- 内容介绍常用 Markdown 语法，让用户直接在预览面板看到渲染效果

#### F1.8 响应式设计
- 桌面端（>768px）：显示完整三栏布局
- 移动端（<=768px）：隐藏侧边栏，提供切换按钮显示/隐藏侧边栏；编辑器和预览可切换显示

#### F1.9 深色主题
- 默认使用深色主题编辑器风格
- 编辑器背景深色、文字浅色，预览面板同样适配深色

## Milestone 2: 增强功能

**状态**: 已完成 (2026-03-16 验收通过)

### 功能清单

#### F2.1 笔记搜索
- 侧边栏顶部搜索框
- 支持标题和内容全文搜索
- 输入时实时过滤笔记列表
- 搜索为空时显示全部笔记

#### F2.2 笔记标签系统
- 每个笔记可以添加多个标签
- 编辑器上方或笔记列表项中显示标签
- 侧边栏提供标签筛选功能（点击标签过滤笔记）
- 标签支持创建和删除

#### F2.3 编辑器工具栏
- 编辑器上方提供格式化快捷按钮
- 包含：加粗、斜体、标题（H1-H3）、链接、代码、无序列表、有序列表
- 点击按钮在光标位置插入对应 Markdown 语法

#### F2.4 字数统计
- 编辑器底部实时显示：字符数、单词数、行数
- 随输入实时更新

#### F2.5 导出单个笔记
- 每个笔记提供"导出"操作
- 导出为 `.md` 文件下载
- 文件名为笔记标题

#### F2.6 导入 .md 文件
- 提供"导入"按钮
- 支持选择 `.md` 文件导入为新笔记
- 文件名作为笔记标题，文件内容作为笔记正文

#### F2.7 全部笔记备份与恢复
- 导出所有笔记为 JSON 文件（含标题、内容、标签、时间戳）
- 导入 JSON 文件恢复笔记（覆盖或合并策略由实现决定，但需有提示）

#### F2.8 快捷键
- `Ctrl+N`（Mac: `Cmd+N`）：新建笔记
- `Ctrl+S`（Mac: `Cmd+S`）：手动保存（并显示保存提示）
- `Ctrl+F`（Mac: `Cmd+F`）：聚焦搜索框

| AC2.2 | 可为笔记添加/删除标签，侧边栏可按标签筛选 | 操作验证 |

## 技术约束

1. **纯前端**：仅 HTML + CSS + JS，不使用任何前端框架（React/Vue/Angular 等）
2. **无后端**：所有数据存储在 localStorage
3. **CDN 依赖允许**：可通过 CDN 引入 Markdown 解析库（如 marked.js）和代码高亮库（如 highlight.js）
4. **文件结构**：严格保持 `index.html`, `style.css`, `app.js` 三个文件
5. **部署**：Vercel 静态部署

## 设计要求

- 深色主题为主色调
- 编辑器使用等宽字体
- UI 简洁，无多余装饰
- 交互响应迅速，无明显延迟
