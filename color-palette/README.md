# Color Palette Generator - PRD

## 项目概述

交互式颜色调色板生成器 Web 应用。用户输入一个基础颜色，应用自动生成多种配色方案，帮助设计师和开发者快速获取和谐的颜色组合。

- **项目类型**: 纯前端 Web 应用（HTML + CSS + JavaScript，无框架依赖）
- **代码目录**: `projects/color-palette/`
- **部署方式**: Vercel 静态站点
- **在线演示**: https://color-palette-pearl.vercel.app

## 用户画像

- 前端开发者：需要快速获取配色方案用于项目
- UI/UX 设计师：需要探索颜色搭配灵感
- 初学者：想了解色彩理论（互补色、类似色等）

## 用户交互流程

1. 用户打开页面，看到一个颜色选择器（支持 HEX 输入和原生 color picker）
2. 选择/输入一个基础颜色后，页面自动生成 5 种配色方案
3. 每种方案以色块卡片形式展示，附带颜色代码
4. 用户点击任意色块，颜色代码复制到剪贴板，显示成功提示
5. 用户可切换颜色格式、导出方案、收藏方案等

## 技术要求

- 纯前端：HTML + CSS + JavaScript，不依赖任何框架或库
- 颜色计算使用 HSL 色彩空间
- 响应式设计，支持手机和桌面端
- 所有代码放在 `projects/color-palette/` 目录下
- 入口文件为 `index.html`

## Milestone 规划

### Milestone 1: 基础调色板生成 [当前]

**目标**: 实现核心的颜色选择和配色方案生成功能。

**功能清单**:

1. **颜色选择器**
   - 提供 HEX 输入框（支持手动输入如 `#FF5733`）
   - 提供原生 `<input type="color">` 色盘
   - 两者实时联动：改一个，另一个同步更新

2. **配色方案生成**（基于 HSL 色彩空间计算）
   - **互补色 (Complementary)**: 色相旋转 180 度，生成 2 色（原色 + 互补色）
   - **类似色 (Analogous)**: 色相 +/- 30 度，生成 5 色
   - **三等分色 (Triadic)**: 色相等分 120 度，生成 3 色
   - **分裂互补色 (Split-Complementary)**: 互补色两侧各偏 30 度，生成 3 色
   - **单色 (Monochromatic)**: 同色相，不同明度/饱和度，生成 5 色

3. **色块展示**
   - 每种方案独立一个区域，有方案名称标题
   - 每个色块显示颜色预览 + HEX 值
   - 色块大小适中，色彩显示直观

4. **复制到剪贴板**
   - 点击色块/HEX 值，自动复制 HEX 到剪贴板
   - 显示短暂的"已复制"成功提示（toast 或 tooltip 形式）

5. **响应式布局**
   - 桌面端：方案卡片横向排列或网格布局
   - 移动端：方案卡片纵向堆叠
   - 色块在各尺寸下均可正常显示和点击

**验收标准**:

- [ ] 页面加载后显示颜色选择器，默认有一个初始颜色
- [ ] 修改颜色后，5 种配色方案自动刷新
- [ ] HEX 输入框和 color picker 实时联动
- [ ] 每种方案的色块数量正确（互补 2 色、类似 5 色、三等分 3 色、分裂互补 3 色、单色 5 色）
- [ ] 点击色块可复制 HEX 到剪贴板，有成功提示
- [ ] 在 320px~1920px 宽度下页面布局合理，无溢出或重叠
- [ ] 纯 HTML+CSS+JS 实现，无第三方依赖
- [ ] 颜色计算基于 HSL 色彩空间

### Milestone 2: 增强功能

**目标**: 增加实用工具和个性化功能，提升应用价值。

**功能清单**:

1. **颜色格式切换**
   - 支持 HEX / RGB / HSL 三种格式
   - 全局切换按钮，一键切换所有色块的显示格式

2. **调色板导出**
   - 复制为 CSS 变量格式（如 `--color-1: #FF5733;`）
   - 复制为 JSON 格式
   - 导出按钮放在每个方案卡片上

3. **收藏调色板**
   - 收藏按钮保存当前配色方案到 localStorage
   - 收藏列表页展示所有保存的方案
   - 支持删除收藏

4. **随机生成**
   - Random 按钮随机生成一个基础颜色
   - 触发所有方案重新计算

5. **WCAG 对比度检查**
   - 显示两色之间的对比度比值
   - 标注是否满足 WCAG AA/AAA 标准
   - 以直观方式展示（绿色通过/红色不通过）

6. **深色/浅色模式切换**
   - 页面支持 dark/light 主题
   - 切换按钮在页面头部
   - 用 CSS 变量实现主题切换

**验收标准**:

- [ ] 颜色格式可在 HEX/RGB/HSL 之间切换，色块显示格式跟随变化
- [ ] 可导出为 CSS 变量和 JSON 格式，内容正确
- [ ] 收藏功能使用 localStorage，刷新页面后收藏仍存在
- [ ] 随机按钮每次生成不同颜色
- [ ] 对比度检查数值正确，WCAG 标注准确
- [ ] 深色/浅色模式切换正常，视觉效果良好
