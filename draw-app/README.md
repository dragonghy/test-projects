# Draw & Paint - 浏览器端绘画工具应用

**在线演示**: https://draw-app-omega.vercel.app

**部署状态**: ✅ 已部署到 Vercel

## 产品目标

为想要快速画草图、简单插画或像素画的用户提供一个轻量、易用的浏览器端绘画工具。

**核心价值**：打开即用，零学习成本，覆盖日常绘画需求。

## 技术约束

- **纯前端 Web 应用**：HTML + CSS + JS，不使用框架
- **部署方式**：Vercel 静态部署
- **代码目录**：`projects/draw-app/`
- **文件结构**：`index.html`, `style.css`, `app.js`

## 用户交互设计

### 核心交互流程

1. 用户打开应用 → 看到深色主题界面：左侧/底部工具栏 + 中央大画布
2. 默认选中画笔工具，用户可直接在画布上绘制
3. 通过工具栏切换工具（画笔、橡皮、形状等）
4. 通过颜色选择器和粗细滑块调整参数
5. 随时可撤销/重做操作
6. 完成后导出为 PNG 下载

### 布局设计

- **桌面端**（宽度 > 768px）：工具栏在左侧垂直排列，画布占据剩余空间
- **移动端**（宽度 <= 768px）：工具栏在底部水平排列，画布占据上方空间
- 画布自适应窗口大小

### UI 风格

- 深色主题（暗色背景 + 亮色图标/文字）
- 现代简洁风格
- 工具图标清晰易辨认
- 当前选中工具有明显高亮状态

## Milestone 1：基础绘画功能

### 功能清单

| # | 功能 | 优先级 | 说明 |
|---|------|--------|------|
| 1.1 | Canvas 画布 | P0 | 自适应窗口大小，白色背景 |
| 1.2 | 画笔工具 | P0 | 自由绘制，支持不同粗细，笔触流畅 |
| 1.3 | 橡皮擦工具 | P0 | 擦除画布内容，粗细跟随当前设置 |
| 1.4 | 颜色选择器 | P0 | 预设色板（至少 8 色）+ 自定义颜色输入（支持 `<input type="color">`） |
| 1.5 | 画笔粗细调节 | P0 | 滑块控制，范围 1-50px，实时预览当前粗细 |
| 1.6 | 形状工具 | P1 | 矩形、圆形、直线三种，绘制时显示预览 |
| 1.7 | 清空画布 | P1 | 点击后弹出确认提示，确认后清除全部内容 |
| 1.8 | 撤销/重做 | P0 | Ctrl+Z 撤销 / Ctrl+Y 重做，至少支持 20 步历史 |
| 1.9 | 导出为 PNG | P1 | 点击按钮将画布内容下载为 PNG 文件 |
| 1.10 | 响应式设计 | P1 | 桌面端工具栏在侧边，移动端在底部 |
| 1.11 | 深色主题 UI | P1 | 现代深色主题，工具栏和控件统一风格 |

### M1 验收标准

- [x] **AC-1.1** 画布：打开页面后可见白色画布，画布随窗口大小变化自适应调整
- [x] **AC-1.2** 画笔：选择画笔工具后，在画布上按住鼠标拖动可自由绘制线条，笔触连贯无断裂
- [x] **AC-1.3** 橡皮擦：选择橡皮擦后，在画布上拖动可擦除已绘制的内容
- [x] **AC-1.4** 颜色：可从预设色板中点选颜色，也可通过自定义颜色输入选择任意颜色，切换颜色后画笔颜色立即更新
- [x] **AC-1.5** 粗细：拖动滑块可改变画笔/橡皮擦粗细，粗细变化实时可见
- [x] **AC-1.6** 形状：选择矩形/圆形/直线工具后，在画布上拖拽可绘制对应形状，绘制过程中有预览效果
- [x] **AC-1.7** 清空：点击清空按钮后出现确认对话框，确认后画布内容全部清除
- [x] **AC-1.8** 撤销/重做：绘制内容后按 Ctrl+Z 可撤销上一步操作，按 Ctrl+Y 可重做；至少支持 20 步历史记录
- [x] **AC-1.9** 导出：点击导出按钮后，浏览器下载一个 PNG 文件，内容与画布一致
- [x] **AC-1.10** 响应式：桌面端（>768px）工具栏在左侧，移动端（<=768px）工具栏在底部
- [x] **AC-1.11** 深色主题：界面采用深色背景，文字和图标清晰可读，当前选中工具有高亮

## Milestone 2：增强功能

### 功能清单

| # | 功能 | 优先级 | 说明 |
|---|------|--------|------|
| 2.1 | 填充工具 | P1 | Flood fill，点击区域填充当前颜色 |
| 2.2 | 文字工具 | P1 | 点击画布位置后输入文字，可设置字体大小 |
| 2.3 | 图层支持 | P1 | 至少 3 层，可切换当前层，可切换各层可见性 |
| 2.4 | 吸色器工具 | P2 | 点击画布上的颜色，自动设为当前颜色 |
| 2.5 | 网格显示 | P2 | 开关控制，显示像素辅助网格 |
| 2.6 | 快捷键支持 | P1 | B=画笔, E=橡皮, L=直线, R=矩形, C=圆形, G=网格, 1-3=图层 |
| 2.7 | 画布缩放 | P2 | 滚轮缩放 + 适配窗口按钮 |

### M2 验收标准

- [x] **AC-2.1** 填充：选择填充工具后，点击画布上的封闭区域，该区域被当前颜色填充
- [x] **AC-2.2** 文字：选择文字工具后，点击画布位置弹出文字输入，确认后文字渲染到画布上
- [x] **AC-2.3** 图层：界面显示图层面板（至少 3 层），可点击切换当前编辑层，可切换各层的可见性（眼睛图标），不同层的内容互不干扰
- [x] **AC-2.4** 吸色器：选择吸色器后，点击画布上任意位置，当前颜色更新为该位置的颜色
- [x] **AC-2.5** 网格：点击网格开关后，画布上显示辅助网格线；再次点击关闭
- [x] **AC-2.6** 快捷键：按 B/E/L/R/C/G/1/2/3 键可快速切换对应工具或图层
- [x] **AC-2.7** 缩放：滚轮可缩放画布，有"适配窗口"按钮恢复默认缩放
