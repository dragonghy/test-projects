# Flashcard - 闪卡学习应用 PRD

## 项目概述

Flashcard 是一个轻量级的 Web 闪卡学习应用，类似 Quizlet/Anki 的简化版。用户可以创建卡组、添加闪卡，通过翻转卡片进行学习和自测。

- **项目类型**: 纯前端 Web 应用（HTML + CSS + JS，无框架）
- **部署方式**: Vercel 静态部署
- **在线演示**: https://flashcard-steel-two.vercel.app
- **代码目录**: `projects/flashcard/`
- **文件结构**: `index.html`, `style.css`, `app.js`

## 目标用户

- 学生、语言学习者、备考人员
- 需要高频重复记忆知识点的用户

## 核心体验

用户打开应用后看到卡组列表。点击卡组进入学习模式，逐张浏览闪卡，点击卡片翻转查看答案。操作流畅、动画自然、移动端友好。

## Milestone 1: 基础闪卡功能

**目标**: 交付一个完整可用的闪卡学习应用，具备卡组管理、闪卡管理、学习模式等核心功能。

### 功能需求

#### 1. 卡组管理 (Deck Management)
- 创建卡组：输入名称和描述
- 编辑卡组：修改名称和描述
- 删除卡组：确认后删除（连同内部所有闪卡）
- 卡组列表：首页显示所有卡组，每个卡组显示名称、描述、闪卡数量

#### 2. 闪卡管理 (Card Management)
- 在卡组内添加闪卡：输入正面（问题）和背面（答案）
- 编辑闪卡：修改正面和背面内容
- 删除闪卡：确认后删除
- 闪卡列表：在卡组详情页显示所有闪卡

#### 3. 翻转动画 (Flip Animation)
- 点击卡片时触发 3D 翻转动画（CSS transform: rotateY）
- 动画流畅自然，持续时间约 0.6s
- 正面显示问题，背面显示答案
- 再次点击翻回正面

#### 4. 学习模式 (Study Mode)
- 从卡组进入学习模式
- 逐张浏览闪卡，点击翻转查看答案
- 上一张/下一张导航按钮
- 支持键盘操作（左右箭头切换，空格翻转）

#### 5. 进度显示
- 显示 "当前第 X 张 / 共 N 张"
- 进度条可视化

#### 6. 数据持久化
- 所有数据使用 localStorage 存储
- 页面刷新后数据不丢失
- 数据结构清晰（JSON 格式）

#### 7. 响应式设计
- 桌面端：卡片居中展示，合理利用空间
- 移动端（< 768px）：全宽布局，触摸友好
- 最小支持宽度 320px

#### 8. UI 设计
- 现代清爽风格，浅色主题
- 卡片有轻微阴影和圆角
- 操作按钮有明确的视觉反馈（hover/active 状态）
- 适当使用过渡动画提升体验

### 验收标准 (Milestone 1)
- [ ] 可以创建、编辑、删除卡组
- [ ] 可以在卡组内添加、编辑、删除闪卡
- [ ] 点击闪卡有流畅的 3D 翻转动画
- [ ] 学习模式可逐张浏览，支持翻转和上下张导航
- [ ] 显示学习进度（第 X/N 张）
- [ ] 刷新页面后数据不丢失（localStorage）
- [ ] 移动端和桌面端均可正常使用
- [ ] UI 清爽现代，无明显视觉问题

## Milestone 2: 增强功能

**目标**: 在基础功能上增加测验、统计、搜索等进阶功能，提升学习效果。

### 功能需求

#### 1. 测验模式 (Quiz Mode)
- 随机出题，逐题作答
- 用户可选择 "正确" / "错误" 自评按钮
- 答完所有卡片后显示得分统计（X/N 正确，正确率 XX%）
- 可查看错误的卡片列表

#### 2. 学习统计 (Study Statistics)
- 记录每张卡的正确/错误次数
- 卡组详情页显示整体掌握程度（百分比进度条）
- 单张卡片标记掌握状态（已掌握/未掌握/学习中）

#### 3. 乱序/顺序切换
- 学习模式和测验模式可选择随机顺序或原始顺序
- 切换开关在学习界面顶部

#### 4. 搜索功能
- 卡组内搜索闪卡（按正面或背面内容）
- 实时过滤，输入即搜

#### 5. 导入/导出
- 导出：将卡组数据导出为 JSON 文件下载
- 导入：从 JSON 文件导入卡组数据
- 导入时检查数据格式有效性

#### 6. 快捷键支持
- 空格键：翻转当前卡片
- 左/右箭头：上一张/下一张
- 数字键 1-5：测验模式评分（如适用）
- Escape：退出学习/测验模式

### 验收标准 (Milestone 2)
- [ ] 测验模式可随机出题，用户可自评正确/错误，显示最终得分
- [ ] 每张卡有正确/错误统计，卡组显示掌握程度
- [ ] 可切换乱序/顺序浏览
- [ ] 可在卡组内按内容搜索闪卡
- [ ] 可导出 JSON、可从 JSON 导入卡组
- [ ] 快捷键操作正常
