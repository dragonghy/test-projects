# DevToolBox

Free online developer tools collection. SEO-optimized, fast, privacy-friendly.

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- 100% client-side, zero backend

## Features

- 15 developer tools across 3 categories (Data Format, Text & Code, Generators)
- Real-time search/filter
- Dark/light theme toggle
- SEO infrastructure: per-page meta tags, JSON-LD, auto-generated sitemap.xml, robots.txt
- Responsive design (mobile + desktop)
- Reusable ToolLayout component with related tools

## Run Locally

```bash
npm install
npm run dev
# Open http://localhost:3000
```
